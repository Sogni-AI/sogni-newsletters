ü§ñNew Supernet prompt options and Fast Worker advanced configuration ü§ñ


We continue to optimize the platform contests to keep them fair and fun. This season we made the Fast Worker leaderboard more competitive while adding some power user features!


First is the new Fast Worker configuration. Fast Workers now have much more fine-grain control on which models they want to host and rotate between. See the new configuration options in our documentation.


Artists: If you love the reliability of a certain worker and want to help support them, you can now prompt images directly to their machines using the new prompt flag below! Doing 16x batch jobs but a single worker is taking twice as long as the others? You can now exclude them by name in your prompt as well!


Fast Workers: Are your GPUs not getting enough work? You now have the levers to change that and further compete on the worker leaderboard. Send your worker username to friends and family and encourage them to render against your GPUs on Sogni!


‚úÖ Fast Workers can now be targeted by adding --workers=WORKER_USERNAME_OR_NFT_TOKEN_ID directly to your image prompt. All workers who match the username are targetted. You can use a single em dash (long dash) instead of 2 short dashes. You can pass multiple worker usernames or NFT Token IDs separated by a comma.


‚úÖ Certain Workers can now be avoided using a --skip-workers= version of this flag.


‚úÖ Fast Worker and Relaxed Worker now only get leaderboard points for jobs paid for with SOGNI token or Premium Spark. Free Daily Boost use now only credits Artist leaderboard and not Worker leaderboards. This is to combat account and leaderboard farming with Daily Boost.


‚úÖ Fast Workers now get leaderboard points for jobs even from accounts affiliated to that worker. As long as the tokens used are SOGNI or Premium Spark it is now official business! This means you can also build apps on Sogni Client SDK that target just your worker network if you want using the flags above.


‚ùå Both new prompt flags and affiliated worker leaderboard credit currently only work on Fast Supernet. Relaxed Workers will get support soon once network fees are rolled out there to avoid exploitation. "Bad faith" pumping of jobs to your own Relaxed Worker may still result in disqualification from airdrop contests and banning of the accounts.


These flags are currently entered directly into the prompt. In subsequent UI updates we'll be adding user-friendly support in our applications. Right now it is an advanced feature. ü´°
